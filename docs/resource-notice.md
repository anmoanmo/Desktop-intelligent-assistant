# 资源说明

## 资源内容

`Ark-Models-main/` 中包含《明日方舟》相关 Spine 角色模型资源，项目用于演示桌面角色渲染、模型扫描和本地交互效果。

资源目录中的 `models_data.json` 记录了模型数据，`models/` 目录保存角色模型文件。每套 Spine 模型通常包含 `.atlas`、`.skel` 和 `.png` 文件。

## 来源说明

资源来源说明保留在 `Ark-Models-main/README.md`。该目录说明其内容为中国服《明日方舟》中的部分 Spine 模型文件，提取工具为 ArkUnpacker，并注明资源版权归上海鹰角网络有限公司所有。
